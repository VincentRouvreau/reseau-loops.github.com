Le réseau LoOPS a le plaisir de vous proposer une journée sur le thème de l'abstraction matérielle pour vos calculs, avec l'aide de StarPU.

Cette journée, animée par Samuel Thibault de l'Inria de Bordeaux se tiendra le mardi 28 mai 2024 de 10h30 à 17h30 à l'Inria de Saclay - Bâtiment Turing - Salle Henri Poincaré.

Les architectures hardware évoluent rapidement, se complexifient et imposent aux programmeurs de proposer plusieurs versions de leurs codes (multi-coeurs, OpenCL, CUDA, ...). StarPU est une bibliothèque de parallélisation par tâches pour les architectures hybrides (CPU multi-coeurs, GPU, FPGA) qui permet de traiter les dépendances de tâches, l'ordonnancement, les transferts de données et les communications inter-clusters. Votre code source unique développé avec StarPU sera maintenant capable de s'exécuter sur des architectures hétérogènes sans modification.
